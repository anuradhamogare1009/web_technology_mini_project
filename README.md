📝Notes Management System (Mini Project Info)

This project is a web-based Notes Management System that allows users to add, view, and delete notes easily. The frontend is built using HTML, CSS, and JavaScript for creating an interactive user interface. The backend is developed using JavaScript with Express.js to handle API requests.

Instead of using a database, the project stores data in a JSON file, making it simple and suitable for small-scale applications. It follows a REST API structure, where the frontend communicates with the backend to perform operations like adding and retrieving notes.

🎯 Key Features

Add new notes

View all notes

Delete notes

Simple and user-friendly interface

💻 Languages Used in Project

HTML → for webpage structure

CSS → for design and styling

JavaScript → for logic (frontend + backend)

⚙️ Supporting Technology

Node.js → runs backend

Express.js → handles APIs
